# Contributing

Thank you for your interest in contributing to Ethereum Wallet Tracker.

## Getting Started

1. Fork the repository.
2. Clone your fork.
3. Install dependencies:

```bash
npm install
```

4. Create a new branch:

```bash
git checkout -b feature/your-feature-name
```

## Commit Messages

Please use clear and descriptive commit messages.

Examples:

- feat: add wallet history support
- fix: handle invalid wallet address
- docs: update README
- refactor: simplify wallet service

## Pull Requests

- Keep pull requests focused on a single change.
- Update documentation when necessary.
- Ensure the project runs without errors before submitting.
